MapPyMatch 
========================

Mappymatch is a pure-python package developed by the National Renewable Energy Laboratory for matching GPS traces (GPS points captured while transiting) to the route they represent on a maps. The same thing Google maps does when you are using it. Supported matching algorithms LCSS and ____. Supported map formats are OSM and ____. Please see applicable sections for more.
